# 启炼AI 微信小程序 — 页面完善实施计划

## 设计规范（从设计稿提取）
- **主色调**：品牌蓝 #4C8FF9
- **背景色**：#F5F7FA（页面背景）、#FFFFFF（卡片背景）
- **文字色**：#333333（标题）、#666666（正文）、#999999（辅助）
- **圆角**：卡片 16rpx、按钮 48rpx
- **阴影**：0 4rpx 16rpx rgba(0,0,0,0.06)
- **导航栏**：白色背景，黑色标题，左侧返回箭头
- **按钮**：蓝色填充，白色文字，圆角胶囊形

## 待完善页面（5个模板桩页面）

### 1. fitness-test（体能测试）
- 会员信息卡片（头像、姓名、状态、性别、年龄、门店）
- 体能分析报告（运动基础、动作模式、疲劳与恢复、训练表现）
- 雷达图/维度评分展示
- 历史记录列表
- 底部操作按钮

### 2. body-composition（身体成分）
- 会员信息卡片
- 身体成分分析报告（BMI、体脂率、肌肉量、水分等）
- 数据可视化（环形图/柱状图）
- 历史趋势对比
- 历史记录列表

### 3. health-questionnaire（健康问卷）
- 会员信息卡片
- 问卷填写表单（分步骤）
- 问题类型：单选、多选、文本输入
- 进度指示器
- 提交按钮

### 4. joint-screening（关节筛查）
- 会员信息卡片
- 关节活动度评估
- 身体部位图（可点击）
- 各关节评分
- 历史记录列表

### 5. posture-assessment（姿态评估）
- 会员信息卡片
- 体态分析报告（正面、侧面视图）
- 问题标注（高低肩、骨盆前倾等）
- 评分展示
- 历史记录列表

## 实施顺序
1. fitness-test → body-composition → posture-assessment → joint-screening → health-questionnaire
