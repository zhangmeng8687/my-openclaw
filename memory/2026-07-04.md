## 技术笔记

### CSS 响应式
- `position: sticky` 在 flex 布局中可能失效，原因是滚动容器问题
- 解决方案：改用 `position: fixed` 或设置 `height: 100vh + overflow: auto`
- 全局样式 `*, *::before, *::after { box-sizing: border-box; }` 很重要

### Element Plus 样式覆盖
- 使用 `:deep()` 选择器覆盖 scoped 样式
- 或在全局样式文件中覆盖

### SSH 连接
- Windows 上需要用 ssh2 库（Node.js）
- 不能直接用 ssh 命令（需要密码交互）
