# 2026-07-06 工作记录

## 金石康养小程序项目初始化

### 员工端（gold-stone-care-employee）搭建完成
- **app.js**：登录态管理、多机构切换、系统信息初始化
- **utils 层**：
  - request.js：Token 自动注入、401 自动刷新、并发刷新队列、统一错误处理、文件上传
  - storage.js：本地缓存工具（支持过期时间、类型安全、按应用隔离）
  - util.js：日期格式化、防抖节流、UUID、手机号脱敏、弹窗封装
  - config.js：API 基地址、缓存 key、超时时间、分页大小
- **styles/variables.wxss**：CSS 变量（老年友好设计规范）
- **services 层**：
  - auth.js：登录鉴权（微信 code 登录、手机号验证码、Token 管理）
  - user.js：用户服务（档案、机构切换、头像上传）
  - order.js：订单服务（列表、接单、配送、批量操作、核验）
  - live.js：直播服务（推流地址、冲突校验）
- **9 个页面骨架**：登录、首页工作台、机构选择、订单列表、订单详情、抢单大厅、送餐进度、直播推流、个人中心
- **4 个公共组件**：二次确认弹窗、订单卡片、自定义 TabBar、加载状态

### 家属端（gold-stone-care-family）搭建完成
- **app.js**：登录态恢复、老人列表管理、切换老人全局方法
- **utils 层**：request.js、storage.js、util.js、config.js
- **services 层**：
  - auth.js：登录鉴权（微信授权手机号、验证码登录、Token 管理）
  - elderly.js：老人服务（扫码绑定、手机号绑定、解绑、审核状态查询、切换）
- **13 个页面骨架**：登录、首页、绑定老人、餐饮浏览、餐品详情、下单确认、订单列表、订单详情、订单评价、健康信息、商品列表、商品详情、个人中心
- **全局样式**：老年友好（大字号、大按钮、高对比度、圆角卡片）
